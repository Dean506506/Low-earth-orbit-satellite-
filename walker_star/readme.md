物理與軌道常數 (Physical and Orbital Constants)

EARTH_RADIUS = 6371 (km): 這是地球的平均半徑。設定此值是為了將衛星的軌道高度轉換為從地心算起的絕對軌道距離（即軌道半長軸）。

ALTITUDE = 550 (km): 這是衛星在地球表面上方的飛行高度。選擇 550 公里是一個非常典型的近地軌道（LEO）高度，與現代大型衛星星座（如 Starlink 的某個殼層）的運行高度相似，能在通訊延遲和地面覆蓋範圍之間取得良好平衡。

SEMI_MAJOR_AXIS = EARTH_RADIUS + ALTITUDE (km): 這是軌道半長軸，即衛星到地心的距離。它是通過將地球半徑和衛星高度相加得出的，是計算軌道週期最重要的輸入參數。

GRAVITATIONAL_PARAM = 3.986004418e5 (km³/s²): 這是地球的標準重力參數 (μ)。它是一個精確測量的物理常數，對於使用克卜勒定律計算軌道週期至關重要。

EARTH_ROTATION_RATE_DEG_S = 360 / 86164.1 (度/秒): 這是地球的自轉角速度。這裡的關鍵是分母 86164.1 秒代表一個恆星日（地球相對於遙遠恆星自轉一圈所需時間），而不是 24 小時的太陽日。在軌道模擬中必須使用恆星日，因為衛星的軌道在慣性空間中是固定的，而地面軌跡是繪製在自轉的地球表面上。

ORBITAL_PERIOD_S = 2π√(a³/μ) 圓軌道週期公式 它代表一顆衛星環繞地球飛行完整一圈所需的時間 
在您的程式碼設定下，計算出的軌道週期 ORBITAL_PERIOD_S 大約是 5730.5 秒。
